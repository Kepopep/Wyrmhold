#### [[Арканиум]]
> Изучение арканной магии. 
 
Институт походит на более классическую школу магии. Большие библиотеки, зоны для практики, просторные лектории. 

Для исследования наказанных используются пыточные с особыми приборами.  

#### [[Оккультиум]]
> Изучение оккультной магии.

Институт походит психиатрическую лечебницу с решетками на окнах. Большие библиотеки, прогулочные зоны, комнаты с белыми стенами для записей видений, зоны для практики, закрытые комнаты для диалогов вместо больших лекций. 

Для исследования наказанных используются смирительные комнаты, препараты для стабилизации психики, персонализированные инструменты для дестабилизации психики.  
#### [[Праймиум]]
> Изучение первобытной магии. 

Институт походит дендрарий окруженный большими стенами и решеткой сверху. Небольшие библиотеки, прогулочные зоны, область поделена на несколько климатических зон, зоны для практики, открытая поляна для общения. 

Для исследования наказанных используются каменные круги на краю территории, разные пилы и щипцы, персонализированные инструменты для привязки "высаживания" на территории.
